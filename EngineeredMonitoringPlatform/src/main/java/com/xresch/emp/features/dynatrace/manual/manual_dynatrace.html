Dynatrace Widgets Setup

Before the Dynatrace widgets can be used, at least one Dynatrace environment has to be defined by a user with admin permissions. The environments can be added in the Context Settings found under "Admin >> Context Settings". The settings will need the API URL and an API token from the Dynatrace instance.

API Limitations

The API has the following known limitations;

  • SaaS Requests: Dynatrace SaaS solution has a limitation of 50 request/min. Extensive use of Dynatrace widgets can easily reach this limit and will result in error messages. This is not the case with on-premise solutions.
  • Timeframes: Various API calls have limitations on the maximum timeframe they can access. In case such a limit is reached, a warning message is displayed for each widget running into this issue.
