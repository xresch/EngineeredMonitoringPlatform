With this source you can get step data which is cached in EMP. Means you will only get the data which is actually somehow related to a scheduler.

Following returns a list of cached schedulers: 

| source stepdata 
	environment={"id":1234,"name":"STEP Test"} 
	type="schedulers"

Following returns a list of cached plans: 

| source stepdata
	environment={"id":1234,"name":"STEP Test"} 
	type="plans" 
| unbox customFields, attributes 
| remove customFields, attributes

Following returns a list of cached projects: 

| source stepdata 
	environment={"id":1234,"name":"STEP Test"} 
	type="projects" 
| unbox attributes 
| remove attributes
