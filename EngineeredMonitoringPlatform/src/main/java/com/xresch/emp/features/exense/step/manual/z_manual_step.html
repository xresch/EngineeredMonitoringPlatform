Setup

Before the Step features can be used, at least one Exense STEP connection has to be defined by a user with admin permissions. The environments can be added in the Context Settings found under "Admin >> Context Settings". Hover the info icon next to the settings fields to get more info about what you need to fill in. When saving the STEP context setting, all the scheduler will be reloaded, what might take several minutes depending on the number of schedulers you have.

Widgets

Several Widgets are included in the Exense Step feature. Please find their documentation in this manual under section Dashboard >> Widgets >> Exense Step

Widget Status Colors

The status colors displayed are defined as follows:

  • Step Result:  The result of the STEP execution, like RUNNING,PASSED, FAILED etc...
  • Custom Performance Thresholds(optional):  Most STEP widgets provide threshold fields. If the execution has status PASSED thresholds will be evaluated. Define a times in milliseconds that will evaluate the duration of the scheduler execution.
