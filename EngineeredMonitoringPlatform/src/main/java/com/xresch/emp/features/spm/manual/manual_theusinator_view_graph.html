This view is not meant to be used as a monitoring tool but to provide more insights into a single project. In contrast to the other four views, this view can provide data of all measures which are available for the project. The time frame for which data is to be displayed can be chosen from a set of standard time frames, as well as through a date picker which allows for custom time frames. Additionally the time span for which multiple values are being merged into one can be selected from a set of options.

The user can choose between five different chart types:

  • Area Chart
  • Bar Chart
  • Donut Chart
  • Line Chart
  • Pie Chart

The following measures are available:

  • Overall Health
  • Accuracy
  • Availability
  • Page HTML download time (overall response time)
  • Page round trip time (overall response time)
  • Page server busy time (overall response time)
  • Performance
  • Transaction Response Time

The user can choose between the following merge spans:

  • Weekly
  • Daily
  • Hourly
  • Quarter of an Hour

For each of the aforementioned measures, the following data can be retrieved:

  • Average
  • boundCount1
  • boundCount2
  • max
  • min
  • sum
  • sumSquared

The provided standard time frames are:

  • Last Year
  • Last Month
  • Last Week
  • Last 24 Hours
  • Last Hour

Due to server side restrictions, not all time frames are compatible with all merge spans. The following combinations are allowed:

Merge Span Allowed Time
Quarter of an hour Up to two months (60 days)
Hourly Up to three months (90 days)
Daily Up to two years (730 days)
Weekly Up to two years (730 days)

Even though time frames of up to two years are allowed, that does not necessarily mean that there are actually data points available for last two years.
