The SPM Legend Widget shows a legend for the colors used by the other SPM Widgets. The following are the statuses used by the SPM widgets:

  • 100%: The measure is at the maximum possible value.
  • 0 - 99%: The selected measure for the project/monitor has reached the indicated value.
  • No Data: The selected project/monitor did not have any executions in the last 30 minutes, the project/monitor doesn't exist/is inactive, or an error occured while fetching the status from the STM database.
  • Disabled: The widget is disabled by the widget settings.
