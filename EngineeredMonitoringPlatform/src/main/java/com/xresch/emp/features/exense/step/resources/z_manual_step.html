Step Setup

Before the Step features can be used, at least one Step environment has to be defined by a user with admin permissions. The environments can be added in the Context Settings found under "Admin >> Context Settings".

Widgets

Plan Status

The widget takes one or more plans as an input, fetches and displays the last execution status for all selected plans. It also provides threshold settings that will color the tile based on the duration value. If no threshold is provided, the colors are determined by the execution result.
This widget can have a tasks that creates alerts if a certain threshold is reached.

Plan Status by Projects

The widget takes one or more projects as an input, fetches and displays the last execution status for all plan of the selected projects. It also provides threshold settings that will color the tile based on the duration value. If no threshold is provided, the colors are determined by the execution result.
This widget can have a tasks that creates alerts if a certain threshold is reached.

Plan Status by Users

The widget takes one or more users as an input, fetches and displays the last execution status for all plans the selected used has access to. It also provides threshold settings that will color the tile based on the duration value. If no threshold is provided, the colors are determined by the execution result.
This widget can have a tasks that creates alerts if a certain threshold is reached.

Plan Status by Current User

The widget takes the username of the currently logged in EMP user, fetches and displays the last execution status for all plans the STEP user with the same username has access to. It also provides threshold settings that will color the tile based on the duration value. If no threshold is provided, the colors are determined by the execution result.

Plan Status All

This widget fetches and displays the last execution status for all plans in the selected Step instance. It also provides threshold settings that will color the tile based on the duration value. If no threshold is provided, the colors are determined by the execution result.

Plan Executions Last N

The widget takes one or more plans as an input, fetches and displays the last N executions for every selected plans. It also provides threshold settings that will color the tile based on the duration value. If no threshold is provided, the colors are determined by the execution result.
