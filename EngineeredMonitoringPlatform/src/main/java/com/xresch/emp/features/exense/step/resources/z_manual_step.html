Step Setup

Before the Step features can be used, at least one Step environment has to be defined by a user with admin permissions. The environments can be added in the Context Settings found under "Admin >> Context Settings".

Widgets

Plan Status by Projects

The widget takes one or more projects as an input, fetches and displays the last execution status for all plan of the selected projects. It also provides threshold settings that will color the tile based on the duration value. If no threshold is provided, the colors are determined by the execution result.
