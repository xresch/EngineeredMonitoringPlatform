The Prometheus feature adds the prometheus source to the query engine. You can fetch data from Prometheus using the source and further process it with the query engine. Here is a sample query that executes two queries on two different Prometheus environments. You can use Ctrl+Space to get content assist for the environment. 

| source prometheus 
	environment={"id":737,"name":"DEV"} 
	query='increase(jetty_requests_total{instance="localhost:8888"}[interval])' 
| source prometheus 
	environment={"id":929,"name":"TEST"} 
	query='increase(guava_cache_hit_total{instance="localhost:8888"}[interval])' 
| remove job
| formatfield 
	value=['decimals', 2]
	value=['threshold', 0, 20, 40, 60, 80, "bg"]
