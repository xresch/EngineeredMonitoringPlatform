Configuration

Before Theusinator can show any data, you have to provide the URLs and user credentials of the Silk Performance Manager environments.

  1. Create a User in SPM that has access to all projects with Administrator rights.
  2. In EMP, login as an administrator and go to "Admin >> Configuration " and change the settings accordingly.

Users

Create users under "Admin >> Manage Users ". The username has to be the same as in Silk Performance Manager. Theusinator will fetch all the information by username through the SPM API.

Permissions

There are two permissions related to Theusinator. For a user to be able to access the dashboard, he needs a role that has the required permission.
