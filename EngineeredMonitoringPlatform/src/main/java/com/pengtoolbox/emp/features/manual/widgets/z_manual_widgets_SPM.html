SPM Widgets Setup

Before the SPM widgets can be used, at least one SPM environment has to be defined by a user with admin permissions. The environments can be added in the Context Settings found under "Admin >> Context Settings".

Widgets

SPM Legend

The SPM Legend Widget shows a legend for the colors used by the other SPM Widgets.

SPM Project Status

The Project Status Widget will display the status for one or more selected projects. If there is currently no data available, the status will be set to unknown.

SPM Monitor Status

The Monitor Status Widget will display the status for one or more selected monitors. If there is currently no data available, the status will be set to unknown.

SPM Monitor Status for Projects

The Monitor Status for Projects Widget will display the status for all monitors of the project(s) selected. If there is currently no data available, the monitor without data will not be displayed. This widget is useful for projects with a lot of monitors or with frequent changes.

SPM Monitoring All

The Monitor Status All Widget will display the status for all monitors of the selected environment. If there is currently no data available, the monitor without data will not be displayed. This can be used to get a full overview of all monitors for a specific environment.
