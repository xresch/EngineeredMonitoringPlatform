SPM Widgets Setup

Before the AWA widgets can be used, at least one AWA environment has to be defined by a user with admin permissions. The environments can be added in the Context Settings found under "Admin >> Context Settings".

Widgets

AWA Legend

The AWA Legend Widget shows a legend for the colors used by the AWA Job Status Widget. The following are the statuses used by the AWA widgets:

  • Ended OK: The last run of the job ended successfully.
  • Running: The job is currently running.
  • Overdue: The last run of the job is older than the number of minutes specified for the parameter "Last Run Minutes".
  • Issue: The last run of the job ended with an issue.
  • Unknown: The status of the job could not be fetched. Either the job is not found in the AWA/UC4 database, or an exception occured while executing the SQL query.
  • Disabled: The widget is disabled by the widget settings.

AWA Job Status

The Job Status Widget will display the status for one or more selected AWA jobs. If there is currently no data available, the status will be set to unknown.
