Modals can be accessed from each of the four monitoring views:

  • Dashboard
  • Box
  • Health History
  • Status

Clicking on any of the projects (tile or row in Health History View) will open a modal panel with additional information about the project. There are seven tabs, six of which show data for different measures of the project while the seventh provides some facts about the project (project description) and the option to add and remove a custom tag.

The project’s name, the timeframe for which the data is being displayed and three buttons to go to Configuration, Infrastructure and Monitoring (standard SPM views) are always displayed, regardless of the currently selected tab.

Measure Tabs

The six measure tabs are named:

  • Overall Health
  • Availability
  • Accuracy
  • Performance
  • Page Round Trip Time
  • Custom Timers

They all show a table and two charts with data from the last twelve/twenty-four hours, depending on which time frame was selected in the option panel.

The table shows the following measures:

  • Average
  • Maximum
  • Minimum
  • Median
  • 90th Percentile
  • Standard Deviation

All of these measures relate to the averages and not the absolute values. This means that the maximum and minimum represent the extrema of the averages over the last twelve/twenty-four hours, not the actual maximum and minimum values that were recorded. This is the case because the table might otherwise show values which are not represented in the two charts. If a user wishes to see the actual extrema of a project in a specific time frame they can do so in the Graph View.

The chart below the table simply shows the averages of the measure that is currently selected (current tab) over the last twelve/twenty-four hours. The user may choose between the following five chart types:

  • Area Chart
  • Bar Chart
  • Donut Chart
  • Line Chart
  • Pie Chart

The chart in the right half of the modal panel will always show a donut chart with data from the last twelve/twenty-four hours. The user can hover over each of the slices to reveal the timestamp and the corresponding value. An Info icon can also be found in the top left of the right half which explains how to read the donut chart.

Project Description Tab

The Project Description tab features additional information about the project in the form of a table. This table is filled dynamically and depends on the project’s description, which can be set by the SPM Team under “Administration >> Projects”. These are currently set for all projects which were active at the time of writing this guide. If you activate other projects and wish to have the project description show up in the modal like in the image above, do the following:

  1. In Silk Performance Manager go to "Administration >> Projects " and edit the project
  2. Enter a description in this format: 
    {“env”: “yourEnvironment”, “tid”: “yourTID”, “itso”: “yourITSO”, ...}
  3. Save the changes
  4. Activate the project

The custom tag which can be set in the other half of the modal panel will also be listed here. Users can set one custom tag per project which can be between one and eight characters long. These tags can be used for sorting and filtering and will also be visible in the tiles in the Dashboard and Box views.
